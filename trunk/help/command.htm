Command Line Parameters

About

Command Line Parameters are used to further adjust the game behaviour. Under normal circumstances you don't need to set any one of the following parameters, but sometimes it may be necessary due to an old graphics adapter, faulty joystick or similar things. Below is a list of all supported command lines.

To use parameters, start the game from a shell window or create an appropriate desktop shortcut containing something similar to: warriors.exe -wnd -d video7

-wnd

Run game in windowed mode.

-d videoX

Set default resolution. Can also be modified on Options screen in game. Replace X by one of the following numbers:
5 - 512x384
6 - 640x480
7 - 800x600
8 - 1024x768 (default)
9 - 1280x1024
10 - 1600x1200

-d alphadepth

Use 16Bit instead of 32Bit color depth for transparent textures. Required by some old graphics adapters.

-w3d

Run game in compatibility mode. Use this one to get the game running with old hardware, like Voodoo3.

-nj

Disable joystick support. This can be helpful if joystick is not calibrated properly.

-nm

Disable mouse support.

-diag

Create engine log file: acklog.txt

-d wiimote

Use Wiimote® and Nunchuk® as alternative input device and attack by slashing with your arm.
Experimental and unstable implementation only. To set up your Wiimote® to use with PC, please check out wiimote.txt contained in the game folder. Currently only one single device is supported.

Wiimote® and Nunchuk® are properties of Nintendo®.
