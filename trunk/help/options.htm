Game Options

Game Setup

Difficulty - set strength of computer players (Level 1-3)
Time Limit - time to play for eac game (1 to 5 minutes)
No. of Games - games required to win to end game (1, 3 or 5)
Game Speed - speed of player movement (slow, normal, fast)
Default Settings - reset game settings

Input Devices

Input Device - assign input device to player by pressing enter
Control Layout - configure the selected input device
Default Settings - reset input settings

The standard layout is listed under "How to play".
Video Options

Resolution - choose from several screen resolutions
Color Depth - choose between 16Bit and 32Bit color depth
Shadows - use less shadows or blob shadows only to enhance speed on very slow systems
Special Effects - turn special effects off to gain more speed on very old pcs

Audio Options

Music Source - choose between integrated module player, cd playback or no music at all
Sound Volume - set volume for sound effects
Music Volume - set volume for module player (only used when module player is active)
CD Volume - set volume for cd player (only used when cd player is active)
