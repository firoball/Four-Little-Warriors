How to play

Player status


# of crystals - crystals collected this round
Rank - current ranking based on amount of crystals collected
Games won - games already won (only if no. of games is set to higher than 1)
Stamina - required for attacking and blocking, recovers over time
Ammo - ammonition for current weapon, can be refilled by collecting a weapon of the same type

Controls

 Cursor keys - movement
 CTRL - attack/block
 (Assigned to yellow player by default)

 W,A,S,D - movement
 Q - attack/block
 (Assigned to red player by default)

 Mouse move - turn left/right
 Mouse right - forwards
 Mouse middle - backwards
 Mouse left - attack/block
 (Assigned to blue player by default)

 Joystick 1 - movement
 Joy1 button1 - attack/block
 (Assigned to green player by default)

 Joystick 2 - movement
 Joy2 button1 - attack/block
 (Not assigned by default)

(Please note that these are the default settings. Key layout as well as input device assignment for each player can be changed in the options menu.)

Attacking and blocking


Normal attack - hit attack key once
Special attack - when stamina is maxed hit attack key twice
Block - keep attack key pressed (requires stamina)

The stamina amount influences strength of the attack
